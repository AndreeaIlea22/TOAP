Help

Here are some resources to help you with the main Publisher features of DOAJ.

Application Form

Guides to filling in the Application Form are available in other languages. Download the language of your choice and refer to it when filling in the form.

Creating DOAJ XML

For help with creating DOAJ XML and details on XML tags, read the File Upload Info section on our Features pages.

Problems with XML files

For problems with XML file uploads that have failed or have errors, read the Failed XML Uploads section on our 'For Publishers' page.

Correcting articles on DOAJ

If you need to correct an article on DOAJ, upload it to us again. Read How Do I Correct An Error in Article Metadata? on our 'For Publishers' page for more details.

Deleting articles from DOAJ

If an article or a range of articles needs to be deleted, please Contact Us. Please email us ALL of the following information:

  1. Name of the journal, as it appears in DOAJ
  2. The URL of the journal in DOAJ ( which will look something like http://doaj.org/toc/ec02cdfe6de44d4d98be2927640245c8 )
  3. The title of EVERY article to be deleted. If an article appears twice, include details of what is wrong so we know which one to delete.
  4. If you are asking us to delete a range of content, include the volume and issue range. For example, Volume 1, Issue 1, Jan 2010 to Volume 2, Issue 4, Dec 2011.
  5. Send a screenshot if it will help.
