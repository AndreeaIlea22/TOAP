113      don't really use that, it's for things like "total downloads" or

115

117      take into account what previous strings we've sent so that events 
118      can be aggregated correctly. Changing the strings you use for 
119      categories, actions and labels to describe the same event will 
120      split your analytics reports into two: before and after the 
121      change of the event strings you use.

129      record_value_of_which_arg causes.
