batch_lock(type, ids, username, timeout=None)

source code 

Batch lock succeeds and fails as a unit. If locks can't be obtained on everything then all locks are released.

Works by attempting to lock everything, and then backing out, unlocking already locked resources, when it first encounters a locked record
