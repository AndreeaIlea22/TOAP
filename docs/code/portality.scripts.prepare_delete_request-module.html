clean_list(l)

source code 

Clean up a list coming from an HTML form. Returns a list. Returns an empty list if given an empty list.

Example: you have a list of tags. This is coming in from the form as a single string: e.g. "tag1, tag2, ". You do tag_list = request.values.get("tags",'').split(",") Now you have the following list: ["tag1"," tag2", ""] You want to both trim the whitespace from list[1] and remove the empty element - list[2]. This func will do it.

What it does: 1. Trim whitespace on both ends of individual strings 2. Remove empty strings 3. Only check for empty strings AFTER splitting and trimming the individual strings (in order to remove empty list elements).
