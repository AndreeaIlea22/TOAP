Class AssEdJournalReview

source code

Associate Editors Journal Review form. This is to be used in a context where an associate editor (fewest rights) needs to access a journal for review. This editor cannot change the editorial group or the assigned editor. They also cannot change the owner of the journal. They cannot delete, only add notes.
